# Agi-Hybrid-Experiment
“Experimental hybrid AGI system combining transformer models with neuromorphic memory and adaptive learning. This project explores limitations in coherence, training stability, and integration between symbolic and neural reasoning.


AGI Trail (Personal Project) explores a hybrid artificial intelligence system that combines transformer-based language models with a custom neuromorphic architecture 
to simulate memory, adaptive learning, and reasoning. The goal was to move beyond standard conversational models by integrating structured memory and dynamic response 
generation, creating a more flexible and “brain-like” AI system. 
This project was developed independently as an experimental attempt to understand how different AI paradigms can be unified into a single framework.

Despite promising ideas, the system did not fully succeed due to several practical challenges. 
The model faced limited computational resources, which restricted training scale and experimentation. 
It also encountered scalability issues, as integrating multiple complex components made the system difficult to expand or 
optimize efficiently. Additionally, training instability affected performance, leading to inconsistent outputs and coherence breakdowns during conversations
